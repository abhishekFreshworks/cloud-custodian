.. _aws.cfn.actions.delete:

delete
++++++
Action to delete cloudformation stacks

It is recommended to use a filter to avoid unwanted deletion of stacks

:example:

.. code-block:: yaml

        policies:
          - name: cloudformation-delete-failed-stacks
            resource: cfn
            filters:
              - StackStatus: ROLLBACK_COMPLETE
            actions:
              - delete

.. container:: toggle

  .. container:: header

    Schema:

  .. code-block:: yaml

    properties:
      type:
        enum:
        - delete
    required:
    - type


.. _aws.cfn.actions.set-protection:

set-protection
++++++++++++++
Action to disable termination protection

It is recommended to use a filter to avoid unwanted deletion of stacks

:example:

.. code-block:: yaml

        policies:
          - name: cloudformation-disable-protection
            resource: cfn
            filters:
              - StackStatus: CREATE_COMPLETE
            actions:
              - type: set-protection
                state: False

.. container:: toggle

  .. container:: header

    Schema:

  .. code-block:: yaml

    properties:
      state:
        default: false
        type: boolean
      type:
        enum:
        - set-protection
    required:
    - type
