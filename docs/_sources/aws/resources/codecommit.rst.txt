.. _aws.codecommit.actions.delete:

delete
++++++
Action to delete code commit

It is recommended to use a filter to avoid unwanted deletion of repos

:example:

.. code-block:: yaml

        policies:
          - name: codecommit-delete
            resource: codecommit
            actions:
              - delete
