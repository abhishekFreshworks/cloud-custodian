.. _aws.datapipeline.actions.delete:

delete
++++++
Action to delete DataPipeline

It is recommended to use a filter to avoid unwanted deletion of DataPipeline

:example:

.. code-block:: yaml

        policies:
          - name: datapipeline-delete
            resource: datapipeline
            actions:
              - delete

.. container:: toggle

  .. container:: header

    Schema:

  .. code-block:: yaml

    properties:
      type:
        enum:
        - delete
    required:
    - type
