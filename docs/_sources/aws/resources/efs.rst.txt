.. _aws.efs.filters.kms-key:

kms-key
+++++++
Filter a resource by its associcated kms key and optionally the aliasname
of the kms key by using 'c7n:AliasName'

:example:

    .. code-block:: yaml

        policies:
            - name: efs-kms-key-filters
              resource: efs
              filters:
                - type: kms-key
                  key: c7n:AliasName
                  value: "^(alias/aws/)"
                  op: regex
