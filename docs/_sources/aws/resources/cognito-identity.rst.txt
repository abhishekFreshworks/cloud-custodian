.. _aws.identity-pool.actions.delete:

delete
++++++
Action to delete cognito identity pool

It is recommended to use a filter to avoid unwanted deletion of pools

:example:

.. code-block:: yaml

        policies:
          - name: identity-pool-delete
            resource: identity-pool
            actions:
              - delete

    properties:
      type:
